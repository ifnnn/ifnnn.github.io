Linux 终端下的高效工具¶

推荐工具 其他类似 简介
neovim vim 更现代化的 vim
zsh 更好用的 shell
tmux screen / byobu 终端复用工具
fzf 命令行下模糊搜索工具
ag grep / ack 更高效简单的文本搜索工具
tig git 终端下交互管理 git 项目
z autojump / cd 智能模糊搜索, 跳转目录
dstat iostat, vmstat, ifstat 三合一
slurm 网络流量监控
exa ls 检索目录工具
fd find 查找文件和目录
bat cat 带语法高亮的文件查看
ripgrep grep 查找文件中的字符串
jq json 格式化工具
glances htop / top 系统监控工具
axel curl / wget 下载工具，多线程
aria2 wget 下载工具，支持 BT
sz/rz 交互式文件传输，上传下载方便
ranger vim 风格文件管理器
tldr man 缩短说明文档
rsync sftp / scp SSH 协议文件同步
mtr traceroute 网络查询工具
multitail 实时监控多个日志
ccze 高亮显示日志
mycli 交互式管理 mysql
mitmproxy 命令行抓包工具
httpie curl 交互式的 http 客户端
nnn du 文件分析工具
pydf df 磁盘查询工具
asciinema 终端录屏工具
tpp reveal.js 命令行下写 PPT
thefuck 命令行拼写更正
